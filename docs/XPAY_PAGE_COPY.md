## Full description

FailLens Crash Gate turns noisy terminal, build, test, Docker, Playwright, pytest, TypeScript, local-server, and CI failures into compact, redacted failure evidence for coding agents.

Use it when a command fails and produces a huge log. FailLens strips noise, redacts secrets, finds the useful failure window, and returns a safe structured result so agents do not waste context on raw terminal sludge.
