# FailLens Public Proof Examples

These examples are generated from sanitized fixtures.

Each case shows:

- before.md: a compact view of the noisy input log
- after.md: the FailLens safe failure signal
- triage.json: structured JSON output

## Cases

- npm-missing-module: NPM missing module
- playwright-missing-browser: Playwright missing browser
- secret-bearing-crash: Secret-bearing crash
- command-not-found: Command not found
- oom-or-killed: OOM or killed process
- large-noisy-ci: Large noisy CI log
